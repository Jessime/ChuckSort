# ChuckSort

An attempt to capture how one human sorts integers in real life.

## What is this?

This is a toy algorithm that reflects how my dad sorted a list of random numbers written down on paper squares. I explain more about why this is interesting in [this blog post](https://jessimekirk.com/blog/chuck-sort/).
